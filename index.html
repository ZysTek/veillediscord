Présentation

Discord est un logiciel de voix sur IP, comme Skype, conçu principalement pour les communautés de joueurs.

Logiciel multiplateforme créé en 2015, il utilise des technologies Web.

Gratuit pour les utilisateurs, vous pouvez souscrire à un abonnement qui s'appel Discord Nitro. Dans cette article nous allons vous présenter ce logiciel.

Code

Discord à étais codé avec différents langage. Déja pour commencer Discord marche sur Navigateur Web ou par leur Apllication qu'ils ont codé.
Pour le discord qui fonctione par navigateur sont web : HTML , CSS , JavaScript

Pour l'application Discord , elle fonctione avec Elixir , Electron , JavaScript et React (Blibliothéque de JavaScript)

Discord à crée une API pour codé . Sur leur site on trouve L'Api avec de la documentation qui permet d'apprendre les bases.
